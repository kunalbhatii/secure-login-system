🔐 Secure Login System (Flask)
A simple secure login system built using Flask with protection against common web vulnerabilities.
🚀 Features

User Registration & Login
Password hashing
Secure session handling
SQL Injection protection
Vulnerable vs Secure login demo

🛠️ Tech Stack

Python (Flask)
SQLite
HTML/CSS

▶️ How to Run
pip install flask
python app.py

📂 Project Structure

app.py → main app
templates/ → HTML files
create_db.py → database setup

⚠️ Note
Database file is excluded for security reasons.
